# Quantum Learning Thesis Visualization

Interactive educational companion to Allen Liu's 2025 MIT PhD thesis *Learning Theoretic Foundations for Understanding Quantum Systems*.

Hosted on Github Pages: https://sahilohe.github.io/quantum-learning-thesis-viz/

Covers 4 topics:
- **Part 1** — Local Hamiltonians & the Gibbs State: interactive spin grid with temperature slider
- **Part 2** — Sudden Death of Entanglement: phase diagram showing critical temperature threshold
- **Part 3** — Hamiltonian Learning at Any Temperature: parameter recovery simulation
- **Part 4** — Quantum State Tomography & Certification: copy complexity chart
